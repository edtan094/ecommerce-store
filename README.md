# Meme Mart

A full stack Next JS App Router e-commerce application that enables users to purchase and receive memes via email.  

## Technologies Used
* Next JS 14 App Router
* Prisma (ORM)
* PostgreSQL
* Stripe
* Tailwind CSS
* Shadcn components
* Resend for emails
* Vercel Blob Storage
* Vercel for deployments
